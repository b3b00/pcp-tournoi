Mode d'emploi

Un très rapide mode d'emploi pour vous aider à commencer et comprendre mon esprit de développeur :

Première page : sélection ou création d'un tournoi

La première page présente la liste des tournois disponibles et un lien pour en créer un nouveau.

23/09/2019        Gmail - gestion des tournoi

Deuxième page : configuration du tournoi

pour arriver à la deuxième page il faut cliquer sur le bouton Configuration.

La configuration consiste à renseigner tous les élements nécessaires au démarrage du tournoi.

Il y a plusieurs "sous-pages" à parcourir dans l'ordre :

configuration

cette page sert à rensigner les données vraiment basiques :

  1. nom du tournoi : par exemple "tournoi du muguet 2048" ou juste "olivier" pour mes tests

  1. date du tournoi : la date du tournoi : optionel
  2. type de tournoi : simple ou double
  3. nombre de sets gagnants
  4. nom de points par set

le choix du type prérempli les nombres de set gagnant et de points par set . mais on peut ensuite modifier pour mettre 21 points par sets si on veut faire un tournoi années 80 par exemple.

Pour valider cet page cliquer sur le bouton enregistrer

joueurs

Cette page permet d'enregistrer la liste des joueurs et joueuses inscrits

Pour ajouter un joueur il faut renseigner son nom sur la première ligne du tableau. Une case à cocher permet de préciser s'il s'agit d'un joueur licencié.

Enfin pour valider il faut cliquer sur le bouton

le boutons avec la poubelles permet de supprimer le joueur (du tournoi bien sûr pas en vrai :) )

équipes

Cette page permet de constituer les équipes.

Dans le cas d'un tournoi en simple elle ne sert à rien évidemment, d'ailleurs un message vous le dira.

Pour constituer les équipes il faut avoir enregistrer un nombre de joueur pair.

Vous pouvez constituer les équipes de 2 manières :

automatiquement en laissant le logiciel faire avec les 2 boutons :

  • hasard total : le logiciel choisira les 2 joueurs complètement au hasard : ca peut donner des équipes déséquilibrées avec des super forts et des moins forts.
  • 1 licencié - 1 non licencié : ben ca dit clairement ce que ca fait , non ?

manuellement :

pour constituer une équipe : sélectionner 2 joueurs dans la colonne de droite en cliquant sur leur noms : ils apparaissent alors sur fond gris comme sur l'image ci dessus. Puis cliquer sur la flèche vers la gauche pour créer l'équipe.

Enfin vous pouvez modifier / supprimer une équipe en cliquant sur la petite croix en face de chaque joueur dans la colonne de gauche.

Les équipes sont constituées une fois que la colonne de droite est vide.

poules

Permet de constituer les poules... cette page fonctionne de manière très similaires à la page des équipes.

Vous devez d'abord choisir le nombre de poules souhaitées en entrant le nombre dans la zone "nombre de poules" (pour l'instant je ne gére qu'un nombre de poules puissance de 2 : 2 , 4, 8, 16, 32, ... 1024, 2048). puis cliquer sur le bouton "générer les poules".

Les poules sont alors constituées totalement au hasard ce qui peut donner des poules un peu étrange (genre 2 équipes super fortes face à 2 autres trop faibles pour pouvoir espérer passer les la phase de poules)

On peut donc modifier les poules pour les équilibrer : ca fonctionne presque comme pour les équipes :

  1. sélectionner 1 ou plusieurs équipes dans la colonne de droite (elles apparaissent sur fond gris)

  1. sélectionner 1 poules (et 1 seule) dans la colonne de droite en cliquant sur le le nom de la poule (pareil elle apparaît sur fond gris)

  1. cliquer sur la flèche vers la gauche.

Pour retirer une équipe d'une poule, cliquer sur la petite croix à sa droite.

Comme pour les équipes c'est fini quand il ne reste plus d'équipe dans la colonne de droite.

Bravo ! vous avez configuré votre premier tournoi maintenant nous allons pouvoir jouer :)

Troisième page : déroulement du tournoi

Une fois toutes la configuration du tournoi effectuée vous pouvez cliquer sur le bouton Jeu pour accéder au déroulement du tournoi.

Le but de cette page est de réaliser la saisie des scores de tous les matchs pour :

  1. la phase de poules

  1. les tours (8eme quart demi ...)

En cliquant sur le bouton jeu on arrive sur la sous page :

le lien vers tableaux mène aux 2 tableau qui seront créés automatiquement :

un tableau principal : les 2 premiers de chaque poule y sont qualifiés

une consolante : les 2 suivant y sont qualifiés.
